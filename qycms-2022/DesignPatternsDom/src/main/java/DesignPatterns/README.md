# DesignPatternsDom
DesignPatternsDom

java设计模式学习时手敲的实例

每个 package 对应一种模式
## 	创建型模式 Creational Patterns
这些设计模式提供了一种在创建对象的同时隐藏创建逻辑的方式，而不是使用 new 运算符直接实例化对象。这使得程序在判断针对某个给定实例需要创建哪些对象时更加灵活。
### Factory
工厂模式 
### AbstractFactory
抽象工厂模式
### Singleton
单例模式
### Builder
建造者模式
### Prototype
原型模式
## 	结构型模式 Structural Patterns
这些设计模式关注类和对象的组合。继承的概念被用来组合接口和定义组合对象获得新功能的方式。
### Adapter 
适配器模式
### Bridge
桥接模式
### Filter

### Composite

### Decorator
