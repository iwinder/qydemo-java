#阿里云-云市场相关api使用util
  HttpUtils
  
    为阿里云提供的调用api所需的工具类 
	
  pom.xml
  
    为所需的相关依赖。注：未提供完整pom.xml，请将xml中的依赖自行粘贴复制到自己项目中的pom.xml中
  
  ALiApiConfigUtil.java 
    
	api通用配置文件(config)
  
  ALiErrorCoder.java
  
    api通用错误处理文件
